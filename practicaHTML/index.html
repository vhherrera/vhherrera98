Practica HTML

HTML (HypertText Markup Language) en español, (Lenguaje de marcas de hipertexto), hace referencia al lenguaje de marcado para la elaboracion de páginas web. Es un estándar que sirve de referencia del software que conecta con la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código (denominado código HTML) para la definición de contenido de una página web, como texto, imágenes, videos, juegos, entre otros.

HTML se escribe en forma de «etiquetas», rodeadas por corchetes angulares (‹, ›, /). HTML también puede describir, hasta un cierto punto, la apariencia de un documento, y puede incluir o hacer referencia a un tipo de llamado script, el cual puede afectar el comportamiento de navegadores web y otros procesadores de HTML.
